# weather-to-telegram-lambda

[AWS Lambda](https://aws.amazon.com/lambda/) to send [weather](https://wttr.in) to [Telegram](https://telegram.org/) channel via [bot](https://tele.gs/botfather)

[demo](./demo.png)

## Getting Started

Create `.env`

```
CITY=Moscow
TELEGRAM_CHAT_ID=...
TELEGRAM_BOT_TOKEN=...
```

Install

```
$ yarn
$ yarn test
```
